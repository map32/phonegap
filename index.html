What is Phone Gap

An innovative app platform

Phonegap bridges the skills of web development and mobile development by allowing apps to be developed solely through frontend skills.

Only requires JS and HTML

Typical langauge skills required for mobile development - such as Java or Objective-C - are not required.

Compatibility

Phonegap will take care of any compatibility issues. No need to know several different langugages!

Developed by Adobe.

Requirements

Desktop

Download Phonegap Build on your desktop. This will be where all of your coding works are done.

Mobile

Download the Phonegap app on mobile in order to directly test your app on your phone. Apps in development can be put on a server and be connected to your desktop.

Convenient Deployment

Editing the codes on your desktop will immediately take effect on your phone app. This allows for an easy testing without intermittent compiling.

Build the app

After all of the tests are done, your app can be built on build.phonegap.com. It requires an Adobe or a github account.

Advantages

Convenient deployment.
Only requires HTML/JS.
API support of phone features - GPS, camera, etc.

Disadvantages

Slow, especially on canvas games.
Alternative platforms like cocoonJS
